Git

  • git status: Проверить состояние репозитория.
  • git add : Добавить файл в область подготовленных изменений.
  • git commit -m "Сообщение": Сохранить изменения в локальный репозиторий.
  • git push: Отправить изменения на удаленный сервер.
  • git pull: Загрузить изменения с удаленного сервера.
